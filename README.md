## Deployment

This repository auto-deploys to the `gh-pages` branch via GitHub Actions when code is pushed to `master`.

If the links return `404`, go to:

- Repository `Settings` → `Pages`
- Under `Build and deployment`, set source to `Deploy from a branch`
- Choose branch `gh-pages` and folder `/ (root)`
