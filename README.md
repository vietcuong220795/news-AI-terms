Simple static legal pages for Terms of Service and Privacy Policy.

- Home: https://vietcuong220795.github.io/news-AI-terms/
- Terms of Service: https://vietcuong220795.github.io/news-AI-terms/terms.html
- Privacy Policy: https://vietcuong220795.github.io/news-AI-terms/privacy.html

## Deployment

This repository auto-deploys to the `gh-pages` branch via GitHub Actions when code is pushed to `master`.

If the links return `404`, go to:

- Repository `Settings` → `Pages`
- Under `Build and deployment`, set source to `Deploy from a branch`
- Choose branch `gh-pages` and folder `/ (root)`
